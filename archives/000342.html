Ubiquity command: proxy-set

The 'proxy-set' command makes it easy to toggle the proxy configuration.

Here's a screenshot of proxy-set in action:

Read on to install.



By now, the Ubiquity prompt should have appeared at the top of the page offering you to install the proxy-set command.
If you are used to toggle the proxy settings (see screenshot below) between home and work like me, you can now use proxy-set instead. The four options are off, manual, auto and pac (proxy auto configuration file).

Let me know if you have some suggestions or contributions for this command. My to-do list includes improving the preview with a graphical interface.
