A few quick tips that help navigate code more efficiently:

  • F12: Go To Definition.
  • Ctrl+F12: Find All References.
  • Ctrl+M, O: Outline the code (Collapse To Definitions). Reverse it with Ctrl+M, P (Stop Outlining).
