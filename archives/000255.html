ACM Queue - Single Page View

Here's my latest Greasemonkey user script: ACM Queue - Single Page View.

Many of the ACM Queue articles are formatted into multiple pages. Even their printer-friendly versions are not single page.
This script fixes that by combining all the pages for a given article into a single view. The resulting view (screenshot) has as much cruft removed as possible, to let you focus on the content.

IGN.com is probably a bit more complex in terms of screen scraping, but it'd be another great candidate for a similar page re-formatter (that would generate single page views for the articles and reviews). Anyone up for it?
